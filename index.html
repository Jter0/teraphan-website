บริการด้าน Server จาก Teraphan.com

บริการติดตั้ง Web Server, Email Server, File Server, Proxy Server, IT Inventory Server รวมถึงบริการดูแลและ Monitor Server โดยผู้เชี่ยวชาญที่มีประสบการณ์กว่า 5 ปี

ทำไมคุณต้องเลือกบริการจาก Teraphan.com?

Teraphan.com มีประสบการณ์ในด้านการให้บริการติดตั้งเซิร์ฟเวอร์ต่าง ๆ มานานกว่า 5 ปี พร้อมทีมงานที่เชี่ยวชาญและมีความรู้ลึกในทุกระบบเซิร์ฟเวอร์ เรามุ่งมั่นที่จะให้บริการที่มีคุณภาพสูงสุดเพื่อให้ระบบของคุณทำงานได้อย่างมีประสิทธิภาพและปลอดภัย

ไม่ว่าคุณจะต้องการติดตั้ง Web Server, Email Server, Proxy Server หรือ IT Inventory Server เรามีความรู้และเครื่องมือที่ทันสมัยในการตั้งค่าเซิร์ฟเวอร์ให้ตรงตามความต้องการของคุณ พร้อมทั้งดูแลระบบอย่างเต็มที่ตั้งแต่เริ่มต้นจนถึงการบำรุงรักษา

Web Server

ให้บริการติดตั้งและตั้งค่าระบบ Web Server:

  • Apache HTTP Server
  • Nginx (เหมาะสำหรับเว็บไซต์ที่ต้องการความเร็วสูง)
  • IIS (สำหรับ Windows Server)
  • Proxy Server (เช่น Squid หรือ Nginx Reverse Proxy)

ราคา: เริ่มต้นที่ 3,000 บาท/ครั้ง

Email Server

ให้บริการติดตั้งและตั้งค่าระบบ Email Server:

  • Postfix
  • Zimbra Collaboration Suite
  • Microsoft Exchange Server
  • Mailcow (สำหรับระบบ Email แบบ Open Source)

ราคา: เริ่มต้นที่ 5,000 บาท/ครั้ง

File Server

ให้บริการติดตั้งและตั้งค่าระบบ File Server:

  • Samba (สำหรับแชร์ไฟล์ข้ามระบบปฏิบัติการ)
  • FTP Server (เช่น FileZilla หรือ ProFTPD)
  • OwnCloud/NextCloud (สำหรับ Cloud Storage ส่วนตัว)

ราคา: เริ่มต้นที่ 4,000 บาท/ครั้ง

Proxy Server

ให้บริการติดตั้งและตั้งค่าระบบ Proxy Server:

  • Squid Proxy Server
  • Nginx Reverse Proxy
  • HAProxy (สำหรับโหลดบาลานซ์)

ราคา: เริ่มต้นที่ 3,500 บาท/ครั้ง

IT Inventory Server

บริการติดตั้งและตั้งค่า IT Inventory Server:

  • Open-AudIT
  • GLPI
  • OCS Inventory

ราคา: เริ่มต้นที่ 4,500 บาท/ครั้ง

ดูแลและ Monitor Server

บริการดูแลและ Monitor Server ด้วยระบบที่ทันสมัยเพื่อการตรวจสอบตลอด 24 ชั่วโมง

  • การตั้งค่าและติดตั้งระบบ Monitoring เช่น Zabbix, Nagios
  • บริการดูแลระบบ Server และการสำรองข้อมูล
  • ตรวจสอบการทำงานของระบบอย่างต่อเนื่องและแก้ไขปัญหา

ราคา: เริ่มต้นที่ 2,500 บาท/เดือน
